📋 ЧТО НУЖНО СДЕЛАТЬ:

┌───────────────────────────────────────────────────────────────┐
│ ШАГ 1: СОЗДАТЬ БОТА (если ещё нет)                          │
└───────────────────────────────────────────────────────────────┘

1. Открой Telegram
2. Найди @BotFather
3. Отправь: /newbot
4. Введи имя: Pulse Assistant
5. Введи username (должен заканчиваться на 'bot'): PulseHelperBot
6. ✅ СОХРАНИ ТОКЕН (выглядит так: 7234567890:ABCdef123456...)

┌───────────────────────────────────────────────────────────────┐
│ ШАГ 2: НАСТРОИТЬ КНОПКУ МЕНЮ                                  │
└───────────────────────────────────────────────────────────────┘

1. Отправь @BotFather: /mybots
2. Выбери своего бота
3. Нажми: Bot Settings
4. Нажми: Menu Button
5. Отправь URL:
   https://pulse-v2-git-main-d999dimon-uis-projects.vercel.app
6. Отправь название кнопки:
   Открыть Pulse

✅ ГОТОВО! Кнопка появится в левом нижнем углу чата с ботом.

┌───────────────────────────────────────────────────────────────┐
│ ШАГ 3: ПРЯМАЯ ССЫЛКА (Direct Link)                            │
└───────────────────────────────────────────────────────────────┘

После настройки бота, прямая ссылка будет:
https://t.me/YOUR_BOT_USERNAME/app

Пример: https://t.me/PulseHelperBot/app

┌───────────────────────────────────────────────────────────────┐
│ ШАГ 4: ПРОВЕРКА                                               │
└───────────────────────────────────────────────────────────────┘

1. Найди своего бота в Telegram (по username)
2. Нажми /start
3. Должно появиться сообщение с кнопкой "🗺️ Открыть карту заданий"
4. Нажми на кнопку меню слева внизу "Открыть Pulse"
5. Приложение должно открыться!

📝 ДОПОЛНИТЕЛЬНЫЕ КОМАНДЫ ДЛЯ @BotFather:

🔧 АВТОМАТИЧЕСКАЯ НАСТРОЙКА (Node.js):

Если хочешь настроить автоматически через API:

1. Установи Node.js
2. Вставь свой токен в telegram-bot.js
3. Запусти: node telegram-bot.js
