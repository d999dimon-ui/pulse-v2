🤖 Pulse Bot Setup

Настройка Telegram бота за 5 минут

📋 Шаг 1: Создай бота

1 Открой @BotFather
2 Отправь команду: /newbot
3 Введи имя: Pulse Assistant
4 Введи username (должен заканчиваться на 'bot'):
PulseHelperBot
5 ✅ СОХРАНИ ТОКЕН (выглядит как: 1234567890:ABCdef...)

⚙️ Шаг 2: Настрой кнопку меню

1 Отправь @BotFather: /mybots
2 Выбери своего бота
3 Нажми: Bot Settings
4 Нажми: Menu Button
5 Отправь URL:
https://pulse-v2-git-main-d999dimon-uis-projects.vercel.app
6 Отправь название кнопки:
Открыть Pulse

🔗 Шаг 3: Прямая ссылка

После настройки, прямая ссылка будет:

https://t.me/YOUR_BOT_USERNAME/app

Пример: https://t.me/PulseHelperBot/app

✅ Проверка

✓ Найди бота по username

✓ Нажми /start

✓ Проверь кнопку меню слева внизу

✓ Открой приложение

📝 Дополнительные команды

Настрой через @BotFather → Bot Settings → Group Commands:

/start - Запустить приложение /help - Помощь /profile - Мой профиль /tasks - Задания
⚠️ Важно:
  • URL должен быть HTTPS
  • Username бота должен заканчиваться на 'bot'
  • Токен никому не показывай!
